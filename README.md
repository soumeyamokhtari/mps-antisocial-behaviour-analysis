
# London Antisocial Behaviour Analysis (Python)

## Overview
This project analyses antisocial behaviour (ASB) incidents recorded by the Metropolitan Police Service (MPS) in London between 01/08/2023 and 31/08/2024.

The aim is to identify:
- The most frequent types of antisocial behaviour
- The most common hours for incidents
- Boroughs with higher numbers of reported incidents

## Dataset
The dataset used is `MPS_Antisocial_Behaviour.csv`, provided as part of a university case study based on real Metropolitan Police data.

## Tools
- Python
- pandas
- matplotlib

## Analysis
The analysis includes:
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Visualisation of ASB patterns by type, time, and borough

## How to Run
Open and run the notebook:
- `notebooks/ASB_analysis.ipynb`
